# AI Cloud Sentinel

AI Cloud Sentinel is a monitoring and observability system designed to assist in identifying abnormal behaviour in cloud-based services. The platform focuses on analyzing operational signals from systems and presenting insights that help engineers detect potential disruptions early.

## Overview
Modern distributed systems generate a large volume of operational data such as logs and system events. Monitoring these signals effectively is essential for maintaining system reliability.

AI Cloud Sentinel provides a monitoring interface that enables visualization of system activity, identification of unusual patterns, and improved visibility into service behaviour.

## Features
- Monitoring of operational signals from cloud services  
- Detection of unusual behaviour patterns in system activity  
- Dashboard for visualizing service health and monitoring insights  
- Structured monitoring workflow for identifying potential disruptions  
- Modular design suitable for evolving monitoring environments

## Technology Stack
- React
- TypeScript
- Vite
- TailwindCSS

## Project Goal
The goal of this project is to explore approaches for improving visibility and monitoring of cloud-based systems while providing a simple interface for observing operational signals.

## Note
This repository contains the interface and supporting components used for monitoring visualization. The system logic and experimental workflows used during development are intentionally simplified in this repository.
